Day 17: Sticky Header Prototype

Layer Opacity, BringToFront() and .onMove

Again keeping it simple today. I wanted to create a sticky header on scroll. Initially I tried to move each header label out of the scroll element as they moved past the top. Turns out it returns a nasty NotFoundError: DOM Exception 8 -- something about the target element not existing. I switched to setting the opacity of each layer to zero as it reached the top and setting the opacity of a hidden fixed background layer to 1. It did the job.

Day 11: Recreating the NYTvr Transition

Coffeescript String Interpolation, Chrome and Safari Diferences

Today I decided to emulate the transition interaction on New York Times' VR app. In a feed of VR experiences, when the user clicks on one of the experiences, the background images takes over the entire screen and the user is presented with the option of viewing it in Google Cardboard or on their Smartphone. In donig this prototype, I learned about string interpolation in Coffeescript and the differences in Chrome and Safair rendering.
